This is hosted at https://sycamorehr.jeffreyarnell.com

This webpage is now managed by current patrons of the congregation and supplements and supports the needs and functions of that congregation. The site is currently hosted at https://provo23rdward.github.io

A persuasion of computer science that I have been interested in for a long time is the subset Bioinformatics. What originated as a Python program (predominantly a command line tool), this React based website allows users to upload a Fasta file (various sequences of Nucleotide Base Sequences) and will write out the results of a variety of selectable tests to a text file. This project can be found at https://bioinformatics.jeffreyarnell.com
